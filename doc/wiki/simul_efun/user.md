## user_mob_data

## SYNOPSIS

    string user_mob_data(string name)

### PARAMETERS

    string name - The user's name.

### RETURNS

    string - The file path for the user's mob data file, or 0 if the input is invalid.

## DESCRIPTION

Returns the file path for the user's mob data file.
